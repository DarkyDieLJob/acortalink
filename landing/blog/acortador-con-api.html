Acortador de links con API REST: guía para devs 2026

Actualizado enero 2026 · 5 min de lectura

Si estás construyendo una app, un bot, o cualquier sistema que necesite generar links cortos programáticamente, necesitás un acortador con API REST. Acá te muestro qué endpoints ofrece Acortalink, cómo autenticar y cuánto cuesta.

Autenticación

La API usa header X-API-Key. Generá tu key desde el dashboard:

curl -X POST https://app.acortalink.com.ar/api/v1/links/ \
  -H "X-API-Key: tu-api-key-aqui" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://ejemplo.com/pagina-muy-larga"}'

Respuesta:

{
  "link": {
    "short_code": "abc123",
    "short_url": "https://app.acortalink.com.ar/s/abc123/",
    "original_url": "https://ejemplo.com/pagina-muy-larga",
    "is_premium": true,
    "clicks": 0,
    "has_seo": false,
    "has_password": false,
    "created_at": "2026-01-15T12:00:00Z",
    "qr_url": "https://app.acortalink.com.ar/qr/abc123/"
  },
  "created": true
}

Deduplicación automática

Si mandás la misma URL dos veces, la API no crea un link nuevo. Devuelve el link existente con "created": false. Esto evita duplicados sin que tengas que manejarlo del lado del cliente.

Webhooks (Business)

El plan Business incluye webhooks para eventos:

  • link.created — cuando se crea un link
  • link.deleted — cuando se elimina un link
  • click — cuando un link recibe un click

Configurá tu endpoint desde el dashboard y Acortalink envía un POST firmado con cada evento.

Preguntas frecuentes

¿Qué acortadores de links tienen API REST?
Acortalink incluye API REST desde el plan Starter ($4.900 ARS/mes) con 1.000 requests/mes. Bitly incluye API desde $29 USD/mes. Dub.co incluye API desde ~$90 USD/mes. TinyURL no tiene API.
¿Cómo autenticar la API de Acortalink?
La API usa autenticación via header X-API-Key. Generá tu API key desde el dashboard en la sección 'API Keys'. Cada request debe incluir el header 'X-API-Key: tu-key-aqui'.
¿Cuánto cuesta la API de Acortalink?
La API está incluida en todos los planes pagos: Starter (1.000 req/mes, $4.900 ARS), Pro (5.000 req/mes, $9.800 ARS) y Business (25.000 req/mes + webhooks, $28.000 ARS).
¿La API tiene rate limiting?
Sí. El rate limiting es mensual: 1.000 requests en Starter, 5.000 en Pro y 25.000 en Business. Cuando alcanzás el límite, la API devuelve HTTP 429 con headers X-RateLimit-Limit y X-RateLimit-Used.
